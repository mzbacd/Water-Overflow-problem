# Water Overflow problem

## Dependencies

- Node v12.14.0

## Installation

```
git clone https://github.com/mzbac/plexus
cd plexus
npm i
npm start
```

## For development

```
npm run start:dev
```

## Testing

```
npm test
```

# Assumptions

- Input will be valid command
- Space complexity of O(R2) is acceptable
